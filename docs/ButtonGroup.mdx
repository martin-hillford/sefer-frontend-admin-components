import { ThemeProvider } from "styled-components";
import { lightTheme } from "../src/themes";

# ButtonGroup

A ButtonGroup is a simple component to group buttons within the ui:

<ThemeProvider theme={lightTheme}>
    <ButtonGroup>
        <Button variant="default" label="default" />
        <Button variant="primary" label="primary" />
        <Button variant="info" label="info" />
        <Button variant="success" label="success" />
    </ButtonGroup>
</ThemeProvider>

If the property `pull="right"` is set that the buttons will be aligned on the right side of the parent:

<ThemeProvider theme={lightTheme}>
    <div style={{width:"100%"}}>
        <ButtonGroup pull="right">
            <Button variant="default" label="default" />
            <Button variant="primary" label="primary" />
            <Button variant="info" label="info" />
            <Button variant="success" label="success" />
        </ButtonGroup>
    </div>
</ThemeProvider>
