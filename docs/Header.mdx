# Header
Header are available in several sizes. Simple change the `variant` prop to change their size.

<Header variant="xx-large">Very, very large</Header>
This header will be rendered as an `h1` html element.

<br/>
<Header variant="x-large">Very large</Header>
This header will be rendered as an `h2` html element.

<br/>
<Header variant="large">Large</Header>
This header will be rendered as an `h3` html element.

<br/>
<Header variant="medium">Medium</Header>
This header will be rendered as an `h4` html element

<br/>
<Header variant="small">Small</Header>
This header will be rendered as an `h5` html element

<br/>
<Header variant="x-small">Extra small</Header>
This header will be rendered as an `h5` html element
